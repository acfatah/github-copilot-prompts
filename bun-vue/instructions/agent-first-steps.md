# Agent First Steps

When editing this repository:

1. Read `package.json`, `tsconfig.json`, and `CLAUDE.md` at repo root.
2. Use `bun` for all commands - never `npm`, `yarn`, or `node`.
3. Check `src/pages/` for feature-based page and route structure.
4. Inspect `src/router/index.ts` for route registration.
5. Review `src/styles/global.css` for theme tokens and color variables.
